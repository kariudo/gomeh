# gomeh #
gomeh is an interface for the Meh.com daily deal site's API.

## Requirements

- API key from meh.com https://meh.com/developers-developers-developers

## Installation ##

### Install the package:
    go get github.com/kariudo/gomeh

### In your code:

    import (
      "github.com/kariudo/gomeh"
      )
